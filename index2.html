发布器使用范例

Js版

  • JavaScript 代码

    WB2.anyWhere(function(W){
	W.widget.publish({
		id : 'standardSelector'
	});
});
    publish

标签版

  • 默认发布器

    Html 代码

    <wb:publish >微博发布器</wb:publish>
    微博发布器

  • 默认文字+默认图片

    Html 代码

    <wb:publish default_text="这是默认的文字" default_image="http://ww4.sinaimg.cn/bmiddle/3ec33750jw1dnsuig10lpj.jpg">微博发布器</wb:publish>
    微博发布器

  • 可定制的发布按钮

    Html 代码

    <wb:publish button_type="gray" button_size="big" button_text="自定义名字按钮" >微博发布器</wb:publish>
    微博发布器

  • 可选的扩展功能(禁用上传图片)

    Html 代码

    <wb:publish toolbar="face,topic" >微博发布器</wb:publish>
    微博发布器
